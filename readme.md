### Installation

requires [Node.js](https://nodejs.org/) v4+ to run.

Install the dependencies  and start the server.

```sh
$ git clone https://github.com/Govar-Dev/bayanat-sarfi.git
$ npm install
$ cd www
$ npm install
$ composer install
$ cd ..
$ npm start
```
